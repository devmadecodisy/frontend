## 🆘 Troubleshooting

### "npm not recognized"
**→** Restart your computer

### "Cannot find module react"
**→** Run `npm install` in project folder

### Development server won't start
**→** Try different port: `npm run dev -- --port 3001`

### Still have issues?
**→** Reinstall: `rm -r node_modules` then `npm install`
